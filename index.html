About This Project

This movie sentiment analysis project uses machine learning to determine if a movie review expresses a positive or negative opinion. The model was trained on the IMDb dataset containing 50,000 movie reviews labeled by sentiment.

The sentiment analysis model uses TF-IDF vectorization and both Logistic Regression and Naive Bayes classifiers to predict sentiment. The model achieved high accuracy in classifying movie reviews.
